About JinjaBun

JinjaBun is a static site generator that leverages Jinja templating and Tailwind CSS to make building static sites fast, efficient, and enjoyable. Easily control page metadata with YAML, and enjoy features like HTML and JS minification, analytics support, a live server with automatic reload, and automatic generation of robots.txt and sitemap.
